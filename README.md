# üßæ T-Minder

**T-Minder** is a smart and efficient app designed to track product expiration dates effortlessly.  
Instead of manually checking each product, the app automatically monitors expiration dates and sends **notifications** when a product is about to expire ‚Äî one month, two weeks, one week, and daily until it actually expires.  

This helps users take quick actions ‚Äî like replacing or removing the product ‚Äî before it expires, saving time and avoiding waste.
